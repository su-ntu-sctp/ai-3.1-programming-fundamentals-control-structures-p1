# Assignment (Optional)

## Brief

Create Java programs that demonstrate your understanding of variables, data types, operators, and user input.

1. **Simple Calculator Program**
   - Create a file called `SimpleCalculator.java`
   - Use Scanner to read two numbers from the user
   - Perform the following operations and display the results:
     - Addition (num1 + num2)
     - Subtraction (num1 - num2)
     - Multiplication (num1 * num2)
     - Division (num1 / num2)
     - Remainder (num1 % num2)
   - Display each result with a descriptive message
   - Example output:
```
     Enter first number: 10
     Enter second number: 3
     Addition: 10 + 3 = 13
     Subtraction: 10 - 3 = 7
     Multiplication: 10 * 3 = 30
     Division: 10 / 3 = 3
     Remainder: 10 % 3 = 1
```
   - Remember to close the Scanner after use

2. **Data Types and Variables Practice**
   - Create a file called `PersonalInfo.java`
   - Declare variables to store the following information about a person:
     - Name (String)
     - Age (int)
     - Height in meters (double)
     - Is a student (boolean)
     - Grade letter (char)
   - Initialize each variable with sample values
   - Print all the information in a formatted way
   - Example output:
```
     Name: Alice
     Age: 20
     Height: 1.65 meters
     Is Student: true
     Grade: A
```
   - Extra: Try using wrapper classes for at least one primitive type and demonstrate boxing/unboxing
